## 简介

业务组件

## 安装

```bash
yarn add @xmh/comp
#or
npm install @xmh/comp --save
```

## 使用

### 示例

```jsx
import { TestComponent } from "@youzan/ecloud-components";

ReactDOM.render(<TestComponent />, mountNode);
```

### 使用 `babel-plugin-import`

将如下部分增加到 `.babelrc` 的 `plugins` 中，可以让打包的时候按需加载组件代码，节省打包体积。

```js
[
  "import",
  [
    {
      libraryName: "@youzan/ecloud-components",
      libraryDirectory: "lib/components",
      style: (name) => {
        return `${name}/style/index.css`;
      },
    },
  ],
];
```
