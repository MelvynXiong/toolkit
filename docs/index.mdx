## 简介

业务组件

## 安装

```bash
yarn add @ioe/toolkit
#or
npm install @ioe/toolkit --save
```

## 使用

### 示例

```jsx
import { TestComponent } from "@ioe/toolkit";

ReactDOM.render(<TestComponent />, mountNode);
```

## 开发

- 开发模式 `yarn dev`，然后打开 [localhost:3000](//localhost:3000)
